## Platform Overview

### System Description
The Agricultural Civic Intelligence Platform is a comprehensive, AI-driven system that provides farmers and rural communities with:
- Multi-language support (10 Indian languages)
- Real-time agricultural intelligence
- Market insights and price optimization
- Government scheme discovery
- Crop disease detection
- Weather forecasting
- Voice and text-based interactions

### Technology Stack
**Backend:**
- FastAPI (Python)
- PostgreSQL (Database)
- Redis (Cache & Celery Broker)
- Celery (Background Tasks)
- Socket.IO (Real-time Communication)
- OpenAI/Anthropic (LLM Services)
- ChromaDB (Vector Database for RAG)

**Frontend:**
- React 18 + Vite
- Redux Toolkit (State Management)
- React Router (Navigation)
- Tailwind CSS (Styling)
- Socket.IO Client (Real-time)
- Chart.js (Data Visualization)
- React Leaflet (Maps)
- i18next (Internationalization)

4. Real-time weather forecasts with agricultural insights
5. Market intelligence with price comparison
6. Government scheme discovery with eligibility checking
7. Crop disease detection with treatment recommendations
8. Real-time notifications (Browser, SMS)

### Admin Features

3. Performance tracking (API, Database)
4. Cache management and optimization
5. LLM service monitoring and metrics
6. Knowledge base (RAG) management

8. SMS campaign management
9. Translation service management
10. Error logging and alerting

7. Responsive design (Tailwind CSS)
8. State management (Redux Toolkit)
9. Code splitting and lazy loading
